# Vusal Aliyev — Portfolio

Persönliches Portfolio-Website von Vusal Aliyev, Data Analyst aus Bamberg.

Live: [aliyev-vusal.github.io](https://aliyev-vusal.github.io)

## Über das Projekt

Statische Portfolio-Website mit mehrsprachiger Unterstützung (Deutsch, Englisch, Azerbaijanisch). Gebaut mit HTML, CSS, Bootstrap und Vanilla JavaScript.

## Funktionen

- Dreisprachige Benutzeroberfläche (DE / EN / AZ)
- Projektgalerie mit Power BI Dashboards
- Kenntnisse & Tools Übersicht
- Lebenslauf-Sektion
- Kontaktformular
- Responsives Design (Mobile-First)

## Tech Stack

- HTML5 / CSS3
- Bootstrap 4
- JavaScript (Vanilla)
- AOS (Animate on Scroll)
- GitHub Pages (Hosting)

## Struktur

```
├── index.html              # Hauptseite (mehrsprachig)
├── assets/
│   ├── css/style.css       # Hauptstylesheet
│   ├── img/                # Bilder & Screenshots
│   └── js/                 # JavaScript-Dateien
└── _config.yml             # GitHub Pages Konfiguration
```
